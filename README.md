# maual-data-cleaning# Manual Data Cleaning Project

## Project Overview
This project demonstrates the process of manually cleaning a dataset using Python and Pandas. The objective is to identify and correct common data quality issues such as missing values, duplicate records, inconsistent formatting, and incorrect data types.

## Dataset
Dataset Name: Air Quality Missing Data.csv

The dataset contains air quality measurements with some missing and inconsistent values that require cleaning before analysis.

## Objectives
- Identify missing values
- Remove duplicate records
- Correct data types
- Standardize text formatting
- Handle outliers
- Prepare the dataset for analysis
